## Getting Started

- Getting a local copy is simple. Follow the steps below

### Install

- Ruby
- Ruby on Rails
- PostgresSQL

### Local Copy

- Clone the project

```
git clone https://github.com/abdessalem1998/hello-rails-back-end.git
```

### Setup

Install gems:

```
bundle install
```

Setup database:

```
rails db:create
rails db:migrate
```

### Usage

Start server:

```
rails server
```

Visit http://localhost:3000/greetings in your browser.
